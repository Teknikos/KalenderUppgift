Instructions

In this project you will find a simple application that displays events on a calendar.

You'll find documentation for the calendar plugin here: https://fullcalendar.io/docs

Here are a couple of activities you could perform:

  • Change the display of events to separate users by color.
  • Add a tooltip to events in the calendar.
  • Add a filter form above the calendar where a user can select from groups and/or users.
  • Add a new view component to the navigation where new events can be added.
  • Add a summary at the bottom of the calendar
    • Per group
    • Per user
    • Per day
    • Per hour
    • Per event type
  • Highlight all events from the same group when hovering over one.
  • Change localization to sv-SE for date presentation.
  • Create interfaces for types used (events, users, groups) and use it in components instead of 'any'
  • Update navigation menu style
    • Add background color
    • Beautify

Pick and choose from the activities above. Feel free to make any other changes you feel are appropriate.

We'll have a joint review of your result.
